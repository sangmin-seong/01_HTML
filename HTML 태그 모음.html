글자 작성 관련 태그

    - 1. 새 파일 만든 후 ! -> tab
         * ! -> tab 사용 시 기본 문서 양식 구현
    - 2. title 태그 제목 변
    - 3. body 태그 내 주제별 / 제목별 h1~h6 사용하여 작
         * 제목별 글 색상지정은 style="color:fff"를 이용
    - 4. 내용 작성 시 각 단어/문장을 강조할 경우
        가. 글자 굵기            
              ① b(bold) : 단순히 굵게 표시
              ② strong : 글자를 굵게 표시, 시스템 상 표시하는 태그            
        나. 기울임            
              ① i(italic) : 단순히 글자의 모양을 기울림
              ② em : 글자를 기울여 표현함            
               * 웹 접근성으로 인해 strong/em을 사용하여 강조를 함!
        다. 형광펜 및 아래선
              ① mark : 형광펜 표시
              ② u : 아래 밑줄
        라. 취소선 및 삭제선            
              ① s : 취소선
              ② del : 삭제선
