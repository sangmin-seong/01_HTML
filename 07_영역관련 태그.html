block 형식 요소

      - 화면(공간, 영역) 수직 분할
        (세로 화면 가로로 잘라 층을 냄, 레고 블럭)
      - 한 층(행) 모두 차지함.
      - width/height 크기 지정 속성 적용 가능
      - block 형식 요소 종류
        h1~h6(제목)
        p/pre(문단)
        hr(수평선)
        div(영역 분할)
        시맨틱 태그(div 태그에 이름을 추가한 태크)

inline 형식 요소

    - 화면(공간 영역) 수평 분할
      (가로화면/ 영역을 세로로 잘라서 열로 분할, 김밥썰기)
    - 작성된 내용만큼의 크기만 차지함(한정적인 크기)
    -크기 적용 속성(width/height) 지정 불가
    -inline 형식 요소 종류
      b(굵게), i(기울임), strong/em(강조)
      mark(형광펜), span(영역 분할)
inline 형식은 행을 세로로 분할 하는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다

block VS inline : 영역 지정 방식의 차이

  • block 형식 : 사각형형태로 영역을 지정
  • inline 형식 : 작성된 내용(content) 단위로 영역을 지정
